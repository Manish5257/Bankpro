## Getting Started

### Prerequisites

- Node.js 18.0 or higher
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/bankpro.git
cd bankpro
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```
